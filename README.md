# pyfeat web

Temporary camera demo for Py-Feat web verification.

## Structure

- `frontend/`: static web page deployed to `C:\web`.
- `backend/`: local API demo server.
- `uploads/`: local captured images, ignored by git.

## Local dev

```powershell
pnpm install
pnpm dev
```

Open `http://127.0.0.1:8000`.
